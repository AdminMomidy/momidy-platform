# Engineering Decisions (Do not change without a reason)

1) Monorepo: Turborepo
2) Frontend apps: Next.js + TypeScript
3) API: NestJS + TypeScript
4) Database: PostgreSQL
5) ORM: Prisma
6) Auth: Auth.js (Admin uses RBAC roles)
7) CI: GitHub Actions (lint/test/build)
8) Local infra: Docker Compose (Postgres)
